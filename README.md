### Project Overview
This project analyzes historical global sea level data to identify trends and predict future sea level rise. Using data analysis and visualization techniques, the project builds predictive models to estimate sea level changes through the year 2050. The project was completyed as part of teh Freecode camp python certification



### Objectives
- Analyze global sea level data from 1880 to present
- Visualize trends using scatter plots
- Build predictive models using linear regression
- Compare long-term vs recent trends (post-2000)
- Forecast sea level rise up to 2050


### Tools and Technologies
- Python
- Pandas
- Matplotlib
- SciPy (Linear Regression)


### Project Tasks
This project completes the following required tasks:
- Import and clean the dataset using Pandas
- Create a scatter plot of sea level vs year
- Generate a line of best fit using all available data
- Generate a second line of best fit using data from year 2000 onward
- Extend both prediction lines to the year 2050
- Label the graph and save it as an image


### Output
The final output is a visualization that includes:
- Historical sea level data (scatter plot)
- Long-term trend line (1880–present)
- Recent trend line (2000–present)
- Prediction of sea level rise through 2050
